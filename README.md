# React App

Esse é um projeto ReactJs

## Inicializando

Instale:

### `npx create-react-app front-end`

Para instalar todas as dependências

### `npm install axios`

Usado para fazer nossar conexões com a API

### `npm install semantic-ui-react semantic-ui-css`
